Tags HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão para a criação de
páginas web. Combinada com o CSS e o JavaScript, ela forma a base da estrutura e do estilo de
praticamente todas as páginas web que vemos hoje na internet. Nesta matéria, vamos explorar
algumas das tags HTML mais importantes e entender como elas são usadas.

Introdução ao HTML

HTML é composto por uma série de elementos, cada um representando diferentes partes de uma
página web. Os elementos HTML são marcados por tags, que são cercadas por colchetes
angulares. Vamos dar uma olhada em algumas das tags mais comuns:


1. "<html>"

A tag "html" é o elemento raiz de um documento HTML e engloba todo o conteúdo da página.

Exemplo:

            
            <!DOCTYPE html>
        <html lang="pt-br">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Tags HTML</title>
        </head>

2."<head>"

A tag contém informações sobre o documento, como metadados, links para estilos e
scripts, e o título da página.


Exemplo:

                <head>
                    <title>Título da Página</title>
                    <meta charset="UTF-8">
                    <link rel="stylesheet" href="styles.css">
                    <script src="script.js"></script>
                  <head>

3."<body>"

A tag contém o conteúdo principal da página, incluindo texto, imagens, links, etc.


Exemplo:

                <body>
                    <h1>Meu Título Principal</h1>
                    <p>Este é um parágrafo de texto.</p>
                    <img src="imagem.jpg" alt="Descrição da Imagem">
                    <a href="https://www.example.com">Link para Example.com</a>
                  <body>
<4."footer">

A tag <footer> em HTML desempenha um papel crucial na estruturação de uma página da
web, fornecendo uma seção dedicada ao rodapé. O rodapé é uma parte importante do layout de
um site, pois oferece informações adicionais e recursos aos visitantes, além de ajudar na
navegação e na compreensão do conteúdo.


No contexto de design de sites, o rodapé é frequentemente utilizado para incluir:

  1. Direitos autorais e informações legais: O rodapé é um local comum para exibir avisos de direitos
    autorais, licenças de uso, termos de serviço e políticas de privacidade.
  2. Informações de contato: Muitas vezes, o rodapé contém informações de contato, como
    endereço de e-mail, número de telefone, endereço físico ou links para formulários de contato.
  3. Links úteis e recursos adicionais: Pode incluir links para páginas importantes do site, páginas de
    política, páginas de ajuda, mapas do site ou links para recursos externos relevantes.
  4. Redes sociais: Links para perfis em redes sociais são comuns no rodapé, permitindo que
    os visitantes se conectem e interajam em outras plataformas.
  5. Informações sobre a empresa ou organização: Seja um logotipo, uma breve descrição da
    empresa, um lema ou informações sobre a missão e os valores, o rodapé pode ser um local
    para destacar a identidade da empresa.

Ao utilizar a tag "<footer>", os desenvolvedores podem organizar esses elementos de forma
semântica, facilitando a acessibilidade e a compreensão do conteúdo pelos mecanismos de busca e
pelos usuários. Além disso, o rodapé pode ser estilizado usando CSS para complementar o design geral
da página da web e garantir uma experiência de usuário consistente e agradável.
